Comandos

man:
Manual
pwd:
Imprime el nombre del directorio actual
cd:
Cambiar de directorios
ls:
Lista de directorios
touch:
Cambia la fecha y hora de un archivo
mkdir:
Crea un directorio
cp:
Copiar archivos y directorios
rm:
Remover archivos y directorios
cat:
Pega archivos e imprime de forma standard
nano:
Edita o cambia un archivo
more:
Filtra partes de un archivo(hacia adelante)
less:
Lo opuesto de "more"
grep:
Busca: lineas que tengan un patron
echo:
Muestra lienas de texto
tail:
Muestra las ultimas lineas de un archivo(tail-f)

Grid System

Bootstrap incluye un sistema de red(grid system) de fluido móvil sensible, que escala adecuadamente hasta 12 columnas a medida que aumenta el tamaño de la pantalla del dispositivo o de la ventana. Incluye clases predefinidas para opciones de diseño fáciles, así como mixins potentes para generar más diseños semánticos.
